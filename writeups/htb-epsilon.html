Overview

En esta máquina "Epsilon", comenzamos con un escaneo completo de puertos para identificar servicios activos. Encontramos un servidor web con un repositorio Git expuesto y un servidor Werkzeug de Python. Explotamos el repositorio Git para obtener credenciales sensibles y la clave secreta de JWT, que nos permitió generar un token para acceder a la ruta /home. Finalmente, localizamos una clave privada SSH filtrada para escalar privilegios y obtener acceso completo al sistema.

Key steps

  1. Reconocimiento y escaneo de puertos con nmap.
  2. Explotación del repositorio Git para obtener secretos y credenciales.
  3. Generación de token JWT para acceder a /home.
  4. Escalada de privilegios usando clave SSH filtrada.

#!/usr/bin/python3

import jwt
from flask import *

app = Flask(__name__)
secret = 'l33th4x0r'

def verify_jwt(token,key):
    try:
        username=jwt.decode(token,key,algorithms=['HS256',])['username']
        if username:
            return True
        else:
            return False
    except:
        return False

@app.route("/", methods=["GET","POST"])
def index():
    if request.method=="POST":
        if request.form['username']=="admin" and request.form['password']=="admin":
            res = make_response()
            username=request.form['username']
            token=jwt.encode({"username":"admin"},secret,algorithm="HS256")
            res.set_cookie("auth",token)
            res.headers['location']='/home'
            return res,302
        else:
            return render_template('index.html')
    else:
        return render_template('index.html')

@app.route("/home")
def home():
    if verify_jwt(request.cookies.get('auth'),secret):
        return render_template('home.html')
    else:
        return redirect('/',code=302)

3. Escalada de Privilegios y Acceso Final

chmod 600 id_rsa

ssh -i id_rsa epsilon@10.10.11.134

sudo -l

Con estos pasos, logramos obtener acceso completo a la máquina Epsilon como usuario y localizar la flag de root. La clave fue identificar el repositorio Git expuesto y extraer secretos del historial de commits, generando un token JWT válido y utilizando la clave SSH filtrada para escalar privilegios.
