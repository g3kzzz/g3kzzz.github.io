About

This project is a fully static site designed for publishing hacking writeups. It runs 100% client-side on GitHub Pages — no databases, no external servers, pure simplicity and performance.

Features

  • Dark & Light mode toggle 🌙☀️
  • Filter writeups by platform, difficulty, and tags
  • Fast client-side search
  • Responsive & mobile-friendly layout
  • Easy content management via JSON + HTML

How to add writeups

  1. Create a new HTML file in writeups/ (e.g., writeups/htb-example.html).
  2. Add metadata in data/writeups.json (title, platform, difficulty, tags).
  3. Commit & push to GitHub. It auto-appears on the site.

G3K Writeups is built for sharing knowledge, documenting exploits, and helping learners grow in the infosec path.
