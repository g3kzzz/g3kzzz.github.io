About

This site is a pure static front-end designed to work perfectly on GitHub Pages. All content is defined via data/writeups.json and individual writeups/*.html files. No databases or external servers are used.

How to add writeups

  1. Create writeups/your-slug.html with the writeup content.
  2. Add metadata to data/writeups.json.
  3. Commit and push.
